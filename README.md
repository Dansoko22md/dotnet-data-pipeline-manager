# .NET Data Pipeline Manager

This project is a desktop application built with .NET that allows users to create, manage, and execute data pipelines (ETL processes) in a simple and intuitive interface.

It is designed to simulate real-world data engineering workflows, including data extraction, transformation, and loading into a database.

## Features

- Create and manage data pipelines
- Import and process CSV files
- Execute ETL jobs
- Real-time logs and execution status
- Data visualization with tables
- Pipeline execution history

## Technologies

- .NET (C#)
- WPF (MVVM architecture)
- Entity Framework Core
- SQLite

## Use Cases

- Data processing and transformation
- Local data pipeline management
- ETL workflow simulation
- Data engineering practice project

## How It Works

1. Import a dataset (CSV file)
2. Apply transformations (e.g., data cleaning, calculations)
3. Load processed data into a database
4. Monitor execution and view results

## Future Improvements

- Scheduling system (cron jobs)
- Integration with external APIs
- Support for multiple data sources
- Dashboard with advanced analytics

## Project Goal

This project is part of a backend and data engineering portfolio, demonstrating practical implementation of ETL pipelines in a desktop environment.
